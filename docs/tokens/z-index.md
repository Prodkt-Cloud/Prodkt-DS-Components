# Z-Index Tokens

Z-indexes are used to stack components in a logical manner.

| Token                        | Value |
| ---------------------------- | ----- |
| `--moyo-z-index-drawer`      | 700   |
| `--moyo-z-index-dialog`      | 800   |
| `--moyo-z-index-dropdown`    | 900   |
| `--moyo-z-index-alert-group` | 950   |
| `--moyo-z-index-tooltip`     | 1000  |
